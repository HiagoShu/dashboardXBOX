# 🎮 Xbox Data Dashboard: Análise de Dados & Tomada de Decisão

Este projeto consiste em um **Dashboard Interativo** desenvolvido para analisar uma base de dados do ecossistema Xbox (vendas de jogos, performance de consoles e engajamento de usuários). O objetivo principal é transformar dados brutos em insights visuais acionáveis, demonstrando competências essenciais para a área de **Administração e Gestão**.

---

## 📌 Sobre o Projeto

Na administração moderna, os dados são o ativo mais valioso para mitigar riscos e maximizar lucros. Este dashboard foi estruturado para simular um cenário real de negócios, onde gestores precisam acompanhar indicadores-chave de performance (KPIs) para direcionar investimentos, campanhas de marketing e controle de estoque.

### Principais Indicadores Analisados:
* **Performance de Vendas:** Faturamento por região e categoria de jogos.
* **Ciclo de Vida do Produto:** Análise de vendas físicas vs. digitais.
* **Engajamento de Clientes:** Avaliação média dos usuários e horas jogadas por título.

---

## 🛠️ Ferramentas Utilizadas: Adaptabilidade Técnica

> **Nota de Solução de Problemas (Troubleshooting):**
> Este projeto foi desenvolvido inteiramente no **Google Planilhas** devido à falta de acesso temporário à licença do Microsoft Excel no momento do desenvolvimento. 
> 
> Como administrador, acredito que **entregar resultados com os recursos disponíveis** é uma competência vital. O desenvolvimento neste ambiente prova minha agilidade e capacidade de adaptação. Vale destacar que possuo **amplo domínio do Microsoft Excel** (incluindo funções avançadas, Tabelas Dinâmicas, Procv/Proch/Xlookup e tratamento de dados), e toda a lógica analítica aplicada aqui é 100% transferível e aplicável ao software da Microsoft.

---

## 💼 Competências Administrativas Demonstradas

Ao construir este projeto, apliquei conceitos práticos que são diretamente transferíveis para a rotina de um departamento administrativo:

1.  **Pensamento Crítico e Resolução de Problemas:** Limpeza e estruturação de uma base de dados desorganizada para torná-la legível.
2.  **Visualização de Dados (Data Storytelling):** Criação de gráficos intuitivos que permitem a qualquer gestor bater o olho e entender a saúde financeira/operacional do projeto.
3.  **Foco em Resultados:** Definição de metas e KPIs claros para mensurar o sucesso de cada segmento de produto.
4.  **Organização e Processos:** Criação de um fluxo lógico de trabalho, dividindo a planilha em: *Base de Dados (Raw Data)*, *Cálculos/Auxiliares (Back-end)* e *Painel Visual (Front-end)*.

---

## 📊 Estrutura e Recursos Aplicados

Para garantir a robustez do dashboard, utilizei os seguintes recursos técnicos:

* **Fórmulas e Funções:** Uso de funções lógicas e de busca (`PROCV`/`VLOOKUP`, `SE`/`IF`, `SOMASES`/`SUMIFS`) para consolidação de dados.
* **Tabelas Dinâmicas:** Para resumir grandes volumes de dados de forma rápida e eficiente.
* **Formatação Condicional:** Para destacar automaticamente metas atingidas (verde) ou pontos de atenção (vermelho).
* **Filtros Interativos (Segmentadores):** Permitindo que o usuário filtre as análises por ano, gênero do jogo ou plataforma com apenas um clique.
